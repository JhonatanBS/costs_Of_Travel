<p>The project is an application for calculating fuel consumption on a trip. The user interacts with three inputs:</p>

<ul>
  <li><strong>Fuel price:</strong> automatically fetched via a fuel API, ensuring the value is always up-to-date.</li>
  <li><strong>Trip distance:</strong> the user enters the total distance to be traveled.</li>
  <li><strong>KM per liter:</strong> the user inputs how many kilometers the car can drive per liter.</li>
</ul>

<p>Based on these inputs, the calculation is done as follows: the total distance is divided by the car's fuel efficiency (KM per liter) to get the amount of liters needed. Then, the total liters are multiplied by the fuel price, displaying to the user the estimated cost of the trip.</p>
